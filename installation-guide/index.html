                  # Installation guide

                  # Introduction

                  This document is a guide for installing EasySolder 7 on a Windows system. Depending on your hardware configuration, the installation process can take as little as 15 minutes.

                  # Installer types

                  The following installers are available

                  1. Full, regular installer that also contains all drivers and libraries
                  2. Lightweight installer that should be used when drivers and libraries are already set up during previous installations

                  # Full installer

                  Use the regular installer on a new Windows or if EasySolder 7 was not installed before on the PC.

                  Contents

                  • EasySolder 7 files
                  • .NET Framework 4.7
                  • C++ Redistributable 2017
                  • Rework station DBL driver (Silicon Laboratories CP210x VCP drivers)
                  • USB camera driver
                  • Sample workspace

                  # Lightweight installer

                  Use the no-driver, lightweight installer on a PC/tablet where EasySolder 7 or the camera/DBL drivers are already installed.

                  Contents

                  • EasySolder 7 files
                  • Sample workspace

                  # Requirements

                  # Hardware

                  Category Minimum requirement
                  Screen resolution 1024 × 768
                  Hard disk space 100 MB
                  Ports 2 × USB2+

                  # Internet connection

                  Internet connection is not needed. All prerequisites are included in the Full installer.

                  # Operating System

                  The application works on the following 64-bit Windows versions

                  • Windows 7 SP1+
                  • Windows 8
                  • Windows 10
