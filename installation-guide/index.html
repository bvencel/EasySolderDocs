                  # Installation guide

                  # Introduction

                  This document is a guide for installing EasySolder 7 on a Windows system. Depending on your hardware configuration, the installation process can take as little as 15 minutes.

                  # Installer types

                  The following installers are available

                  1. Full, regular installer that also contains all drivers and libraries
                  2. Lightweight installer that should be used when drivers and libraries are already set up during previous installations

                  # Full installer

                  Use the regular installer on a new Windows or if EasySolder 7 was not installed before on the PC.

                  Contents

                  • EasySolder 7 files
                  • .NET Framework 4.7
                  • C++ Redistributable 2017
                  • Rework station DBL driver (Silicon Laboratories CP210x VCP drivers)
                  • USB camera driver
                  • Sample workspace

                  # Lightweight installer

                  Use the no-driver, lightweight installer on a PC/tablet where EasySolder 7 or the camera/DBL drivers are already installed.

                  Contents

                  • EasySolder 7 files
                  • Sample workspace

                  # Glossary

                  Term Description
                  Reflow soldering Reflow soldering is a process in which a solder paste (a sticky mixture of powdered solder and flux) is used to temporarily attach one or several electrical components to their contact pads, after which the entire assembly is subjected to controlled heat, which melts the solder, permanently connecting the joint. Heating may be accomplished by passing the assembly through a reflow oven or under an infrared lamp or by soldering individual joints with a hot air pencil.

                  # Troubleshooting

                  # What to do when the application fails to start after install with the message "This application requires one of the following versions of the .Net Framework"?

                  The installer of the application already deploys a .Net Framework. This message means, that during the installation of the application, the .Net Framework encountered an error and was not deployed. Most probably the problem is caused by an unsupported Windows 10 version. The .NET Framework 4.7+ is not supported on:

                  • Windows 10 1507
                  • Windows 10 1511

                  If you're using Windows 10 1507 or 1511 and you want to install the .NET Framework 4.7.2, you first need to upgrade to a later Windows 10 version. Read more here.
