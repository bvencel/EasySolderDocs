          # User manual

          # Introduction

          Dear Customer,

          This document has been created to explain the operating software EasySolder 7. We recommend that all users intensively familiarize themselves with the equipment, the technology and this document, before operating the Martin Rework System. The complexities, tasks and requirements, when faced with actual processes, may differ from those described.

          When specific application questions arise please contact us.

          # Autoprofiler (Top heater)

          Autoprofiling is the process of determining optimal values for top heater profiles.

          Easy Solder profile controls the energy flow to ensure that thermal limits for components, other surrounding devices and the printed circuit boards are not exceeded. The values for the initial profiler procedure are loaded with the selected soldering tool. Therefore, it is recommended not to modify SMD data before having executed the profiler with original settings once. Usually top heater profiler is used for desoldering option. On Expert 04.6 device the pick-up is performed manually when selecting desoldering.

          Steps to execute top heater profiler

          • Arm calibration (available only for Expert 10.6 device)
          • Placement (if soldering is executed)
            • For Expert 04.6 device, placement is done manually
          • Configure underheater manually or using profiler
          • Set Start temperature, End temperature and Air flow for Top heater profile
            • These parameters must be set for each phase of Top heater profile if it is multi-phased
          • Select soldering/desoldering option
            • Usually autoprofiler is used for desoldering.
          • Attach sensor to the chip
            • Sensor 6 is required when performing top heater profiler with Expert 10.6 device.
            • Sensor 2 is required when performing top heater profiler with Expert 04.6 device.
          • Attach the correct nozzle type
          • Select Run Autoprofiler
          • Follow the application flow, check for correct nozzle type
          • Wait for board to cool down, if the application detects that the temperature is above the threshold
          • Adjust offset using the arrows buttons
          • Click Start
          • If there is no pre-temp or pre-time set, the top heater starts hot air flow over the component.
          • At the end of the rise phase, if the target temperature was not reached, the profiler calculates and sets a peak time.
          • After the peak phase is done, the profiler starts to calculate the hold temperature. Melting point and Stop buttons become available.
            • Press Melting point when the temperature on the solder joint reaches 217°C for lead-free solder (or 183°C for leaded solder respectively). The profiler will continue for a small amount of time (max 20 sec.) to ensure the component is uniformly heated.
            • Press Stop if component temperature reaches the target temperature and melting point has been reached. This will stop the heating immediately.
          • At the end of the profile execution, Profile run statistics window opens.
          • The window displays the result of run, start time, duration, max temperature on solder pen and max temperature
          • Save report, Restart profile, Return to profile and Close option are available

          # Additional information

          Both top- and bottom heaters are active while the parameters are being determined. Underheater uses the already determined parameter set.

          Top heater operates with parameter delivered by solder nozzle selection. Easy Solder writes them into corresponding parameter fields when selecting the tool.

          The top heater parameters are Air flow, Start temperature and End temperature. If those parameters are modified profiler software uses the modified values.

          At the end of the rise time the software compares the actual temperature at the component with a set one.

          • If the SMD temperature is smaller than the set value, profiler software calculates a peak time automatically. The profiler process maintains hot gas power at end temperature for the calculated time. This completes the rise phase; the soldering phase begins.
          • If the SMD temperature is equal or is higher than the set temperature the top heater is switched off and goes directly into the soldering phase.

          In the soldering phase the top heater tries to back-up the requested SMD temperature. Top heater power is controlled to achieve the temperature given in SMD data.

          The profiler needs feedback about melting point and requires to press the button when melting point is reached.

          The profiler calculates the hold temperature from the temperature readings of the solder pen feedback during last 20 seconds, before melting point.

          # Profiler (Underheater)

          The feature is used to establish the correct heating parameters for underheater after specifying only a few initial values (Target temperature, Preheat temperature).

          Maximum temperature that can be set for underheater is 210°C, this also serves as a protection for the component. Overheat can produce damage on the chip.

          As an additional protection, the application checks if the required sensors are connected.

          The temperature on the PCB must be under 45°C. If the temperature is equal or higher with this threshold, fans are turned on automatically (applicable for Expert 10.6 device). The temperature is dropped under 45°C, then after five seconds the profiler starts. The cool down phase can be skipped but is not recommended.

          Steps to run profiler

          • Connect sensor 1 or 5 to board
            • Only sensor 1 is available for Expert 04.6 device.
            • Is recommended to place sensor 1 in a darker place of the PCB, avoiding shiny surfaces.
            • When both sensors are connected the application takes in consideration the maximum between the two sensors.
          • Select board
          • Select Configure
          • Click Profiler
          • Select Temperature and Preheat temperature
          • Run profile
            • The application will remind to attach the sensor at profile run start

          Profiler can be accessed from the following places

          • One option is by selecting the board, then Configure button
          • The second option is to select the component profile, then Profiler button

          # Additional information

          Rework system starts and turn the lamps on. The profiler tries to attain the target temperature as quickly as possible using 100% of the available power. When the board temperature is reached, the Underheater switches off. During off time residual heat in underheater lamps maintain the PCB's temperature stable. After this point the software controls power so that the assembly remains at the desired temperature. The profiler analyzes the underheater controls and works out an optimum power value (Hold power), Rise time and Break time.

          # Soldering

          # Create soldering profile

          New component profiles can be created by selecting component and + button under Profiles. The profile configuration window becomes available after Open profile is selected. Underheater and Top Heater profiles are configured separately.

          # Underheater profile

          Depending on the underheater type, Power and Temperature modes are available.

          # Power

          For Power mode values can be completed manually or using Profiler option. If profiler runs before adding component profile, then the values saved after running profiler were added by default to the new profile.

          List of parameters to be completed for power mode profile

          Parameter name Value
          Lamps Available only for Expert 10.6 HV and HXV. The lamps can be customized by using displayed checkboxes. The lamps covered by the hand rest are grayed out automatically.
          Operating mode Power mode
          Hybrid flow Set level for flow of hot air. Common optimal level is 1
          Board temp Set to 150°C by default and it can be changed from Profiler window. The allowed temperature is between 50°C and 210°C
          Pre time Set period of time for which underheater runs in advance. Underheater and Top Heater profiles normally starts simultaneously but they can be out of sync using this parameter.
          Rise/Hold power Rise power is used for power mode. There are no temperature sensors and only power ratings are set in order to achieve the underheater desired temperature. To quickly obtain the desired temperature, the power of the lamps is set to maximum. Hold power determines the continuous energy input into the assembly during holding phase and is expressed as maximum proportion of power available.
          Rise/Break time Rise time represents the duration of heating up phase measured in seconds. Break time must be set as a safety measure because rapid heating without a brief break in energy input would result a temperature overshoot.
          Power drop rate Need to be set because while running underheater and top heater profile in the same time energy can be supplied by top heater. In this case the hold power cannot ensure the constant temperature on the PCB.

          # Temperature

          For temperature mode values are manually completed and sensors are required at profile execution. Temperature mode is available only for Expert 10.6 HV and HXV.

          List of parameters to be completed for temperature mode profile

          Parameter name Value
          Lamps The lamps can be customized by using displayed checkboxes. The lamps covered by the hand rest are grayed out automatically.
          Operating mode Temperature mode
          Hybrid flow Set level for flow of hot air. Common optimal level is 1
          Board temp Set underheater temperature that has to be achieved while executing profile
          Pre temp Preheat temperature represents the temperature that has to be reached by the underheater before the Top Heater profile is executed. When the parameter is set the Underheater and Top Heater profiles are out of sync.

          # Top Heater profile

          • Top heater profile can be single or multiple-phased
          • Phases are added using the + button next to Top Heater title
          • Recommended values are generated by the application after pressing the ↓ button
            • Get recommended function determines the solder pen temperature based on target temperature.
          • Values can be obtained by running Autoprofiler option. In this case Start temperature, End temperature and Air flow parameters has to be defined.
            • Autoprofiler option also optimizes the Peak time, Hold temperature and Hold time.
          • When creating top heater profile, the parameters specified below must be completed. They can be found in the chip data sheet provided by the producer.

          Top Heater profile parameters

          Parameter name Value
          Name Define a meaningful name for the phase or leave generated default
          Component temperature Soldering temperature of the component
          Rise ramp Rise ramp determines how fast the solder pen will reach the end temperature. It is measured in K/s
          Hold time Specifies the time during which the soldering temperature is to be maintained
          Air flow This value specifies the delivered hot air volume. Small components should not be exposed to high air flows. The increased air pressure tends to push the components down, which can lead to electrical short-circuits.
          End temperature End value of the heating ramp that the hot gas source follows to achieve the target temperature of the component
          Hold temperature Holding temperature of the hot gas source to keep the component temperature constant
          Peak time Because of the latent energy of the component on the board it is possible that the component may not have reached the required temperature immediately. To give time to catch up, the end temperature is maintained by the time specified by this parameter.

          # Soldering steps

          When running a profile, the application checks for attached sensors and temperature threshold of 45°C. If the temperature is above the threshold the fans start to cool down the board; when the temperature is below the threshold the profile can be executed. For Expert 04.6 device the Arm calibration is not performed and the Placement is done manually.

          The soldering process requires the following steps

          • Arm calibration
          • Placement
          • Configure underheater and top heater profiles
          • Select soldering option with/without PCB cooling
            • Cooling is available on for Expert 10.6 device
            • When cooling is selected Airflow, Cool comp and Cool board parameters must be set
            • Airflow represents the amount of air used while cooling. Is measured in liter/minutes
            • Cool component represents the time for cooling component. Is measured in seconds
            • Cool board represents the time for cooling the board. Is measured in seconds
          • Connect required sensors
          • Attach the correct nozzle type
          • Select run profile
          • Follow the application flow, check for correct nozzle type
          • Wait for board to cool down, if the application detects that board temperature is above the threshold
          • Adjust offset using the arrows buttons
          • Click Start

          # Desoldering steps

          The desoldering process requires the following steps

          • Arm calibration (available on Expert 10.6 device only)
          • Prepare chip offset
          • Configure underheater and top heater profiles
          • Connect sensors
            • Sensor 1 for underheater
            • Sensor 2 or 6 for top heater on the chip. This depends on the device in use
              • Sensor 2 is available on Expert 04.6 device
              • Sensor 6 is available on Expert 10.6 device
          • Select Desoldering option
          • Select Pickup (if applicable)
          • Select Pickup mode
            • For Expert 04.6 device the pick-up is done manually
          • Run profile
          • Follow the application flow, check for correct nozzle type
          • Wait for board to cool down, if the application detects that board temperature is above the threshold
          • Adjust offset using the arrows buttons
          • Click Start

          At the end of the profile execution, Profile run statistics window opens. The window displays the result of run, start time, duration, max temperature on solder pen, max temperature. Options available at the end of profile run: Save report, Restart profile, Return to profile and Close

          # Reports

          • They can be saved in html or csv format, in a specific folder with the report name specified by the user at saving. The place where the report is saved is also selected by the user.
          • Reports can be opened using web browser.
          • Report name is generated automatically "Profile run report for profile" + profile name
          • Date and hour of report is displayed under the title

          Board Data Board name, Solder type, Underheater type
          Component Data Solder nozzle, Component description

          Cooling phase Enabled, Airflow, Cool component time, Cool board time
          Pickup phase Enabled, Pickup type, Multi pickup

          # Arm calibration

          Arm calibration needs to be executed the first time a lens type is used. The arm calibration process has several purposes:

          • Ensures light and image brightness parameters have optimal values.
          • Measures the arm movement capabilities and maps them to real-world units (millimeters and degrees).
          • Allows the operator to adjust camera alignment.
          • Calculates perspective parameters that ensure accurate component placement.

          Remove PCB before calibration

          Please make sure there is no PCB board installed over the underheater before starting the calibration. During the process the nozzle will drop and execute a few low movements that could hit components on the board.

          The steps performed during arm calibration

          • Auto white balance
          • Auto exposure and gain
          • Drive area
          • Nozzle calibration
          • Nozzle overshoot
          • Camera alignment
          • Perspective
          • Angular calibration

          Notes

          • This operation is available only when connected to a device.
          • For Maxi BGA XL lens, camera calibration has to be performed before arm calibration.
          • The Camera alignment step can only be executed with the BGA lens. For all other lens types this step will be skipped. Therefore, when a rework station is first used after being installed, it is recommended to do an arm calibration with the BGA lens before using any other lenses.
          • Global application settings that affect the calibration process are listed below. See the Settings documentation for more information:
            • Maxi BGA XL lens
            • BGA/QFP 15 nozzle

          Steps to execute arm calibration

          • Select Tools menu
          • Click Arm calibration
          • Click Start
          • Select Continue

          Camera calibration is required only for the Maxi BGA XL lens. This process determines a set of optical parameters (focal length, principal point, distortion) that cannot be measured during the arm calibration process due to the increased image size.

          # Dispensing settings

          Dispensing settings can be accessed from Tools menu. Device must have integrated dispensing feature in order to be able to perform the operation. The integrated dispensing pen of the device allows the precise dispensing of a wide variation of media (SMD adhesive, solder paste, flux cream). Two dispensing options are available

          1. Point dispensing
            • Viscosity and Volume are set
          2. Line dispensing
            • The pressure is set in bar measurement unit

          Steps to perform dispensing

          • Open Tools menu
          • Select dispensing option
          • Set dispensing parameters for the chosen dispensing type
          • Press foot pedal available on the device
          • Use dispensing pen to complete the operation

          # Fans

          Fans can be controlled independently using Tools menu option. Fans are available only on Expert 10.6 device.

          Steps to control fans

          • Open Tools menu
          • Select Fans
          • Click Start/Stop

          # Vacuum

          The vacuum for nozzle and vacuum pen can be controlled independently of component profile section. It can be accessed using Vacuum option from Tools menu. In order to perform the operation, the device and external pressure must be connected.

          # Temperatures

          Temperature monitoring window can be accessed from Tools menu. Sensors connected to device are displayed with their corresponding temperature in Celsius (°C).

          # Application settings

          Application settings window is accessible from Main menu->Settings. The following settings are available:

          • Maxi BGA XL lens
            • This option needs to be selected when using the Maxi BGA XL lens
            • If this option is not set the software will detect the lens as 'Flip-Chip'
          • BGA/QFP 15 nozzle
            • Enable this checkbox for accurate placement when the BGA/QFP 15 nozzle is used
            • It is also recommended to run an arm calibration after changing this setting

          # FAQ

          # How to proceed when connection is lost between PC and device?

          In case of connection loss between device and software, the status is not updated in real time so a software restart is needed.

          # What happens if sensor is disconnected during profile run?

          The application constantly monitors sensors values and if a value decreases abruptly, the profile running is automatically stopped by the application.

          # What happens with the component profiles when the underheater type is changed?

          If underheater type is changed the old underheater profile is lost and a new default profile is generated for the current underheater type. These values have to be adjusted according to current underheater type.
