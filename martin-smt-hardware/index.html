        # Martin-Smt hardware

        # Supported underheater types

        1. Expert 04.6 IXH
          • Underheater lamps not controlled by software
        2. Expert 10.6 HV
          • It has six lamps that are controlled by software
        3. Expert 10.6 HXV
          • It has eight lamps that are controlled by software

        # Supported nozzle types

        1. BGA
        • BGA 23x23
        • BGA 27x27
        • BGA 31x31
        • BGA 35x35
        • BGA 37.5x37.5
        1. QFN
        • CSP/QFN 11x11
        • CSP/QFN 16x16
        1. Micro SMD
        • Micro SMD 7.0/1.1
        1. QFP
        • QFP 20x26
        • QFP 45x45

        # Supported lens types

        1. Maxi BGA XL
        2. Maxi BGA
        3. BGA
        4. CSP
        5. Flip-Chip

        # Sensors

        # Sensors available on devices

        • Sensors available for Expert 10.6
          • Sensor 1 is an infrared sensor attached in the device construction. Is controlled by an on/off buton.
          • Sensor 2 is integrated on the device construction in the left side of the lamps, is represented by a blue light. The application uses it automatically.
          • Sensors 3, 4, 5 and 6 represented by thermocouples connected to the device sensor ports
        • Sensors available for Expert 04.6
          • Sensors 1 and 2 represented by thermocouples connected to the device sensor ports

        # Sensors required when performing operations

        • Sensor 1 -> used for underheater profiler
        • Sensor 5 -> used for underheater profiler
        • Sensor 6 -> used for top-heater profiler
        • The other sensors are optional
